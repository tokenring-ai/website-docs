# Product Manager

**Agent ID:** `productManager`  
**Application:** TokenRing Coder  
**Category:** Planning & Management

## Overview

The Product Manager creates comprehensive Product Requirements Documents (PRDs), defines user stories, and plans feature roadmaps with a focus on user needs and business value.

## Capabilities

- PRD creation and documentation
- User story definition
- Feature roadmap planning
- Requirements gathering
- Stakeholder communication
- Success metrics definition

## Best Use Cases

- New feature planning
- Product requirement documentation
- User story creation
- Feature prioritization
- Roadmap development

## Example Usage

```
> @productManager Create a PRD for a mobile app onboarding flow
> @productManager Define user stories for the checkout process
> @productManager Plan a feature roadmap for Q1 2024
```

## Deliverables

The Product Manager typically provides:
- Detailed PRDs with objectives and requirements
- User stories with acceptance criteria
- Feature specifications
- Success metrics and KPIs
- Implementation priorities

## When to Use

Use the Product Manager when you need:
- Clear product requirements
- User-focused feature planning
- Structured documentation
- Roadmap development
- Stakeholder alignment
