# Product Manager

**Agent ID:** `product-manager`  
**Application:** TokenRing Coder  
**Category:** `Coding - Specialized`

## Overview

Call this agent to analyze requirements and create comprehensive project plans. Provide high-level goals, user needs, or business requirements. The agent will create PRDs, define user stories, plan feature roadmaps, break down tasks, and establish project scope and timelines. Best used for: project planning, requirements analysis, PRD creation, user story definition, and MVP scoping.
