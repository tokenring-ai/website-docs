# Data Engineer

**Agent ID:** `data-engineer`  
**Application:** TokenRing Coder  
**Category:** `Coding - Specialized`

## Overview

Call this agent to implement data migrations, ETL pipelines, and data processing systems. Provide data requirements, migration needs, or processing specifications. The agent will create database migrations, implement ETL processes, design data pipelines, handle data validation, set up data warehousing, and build data processing workflows. Best used for: data migrations, ETL implementation, data pipeline design, data validation, and analytics infrastructure.
