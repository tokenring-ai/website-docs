# Managing Editor

**Agent ID:** `manager`  
**Application:** TokenRing Writer  
**Category:** Writing Agents

## Overview

The Managing Editor coordinates content creation by searching for trending news topics, evaluating newsworthiness, creating article assignments, and dispatching tasks to specialized writing agents.

## Capabilities

- Trending news topic discovery
- Newsworthiness evaluation
- Article assignment creation
- Agent coordination and task delegation
- Editorial planning and strategy
- Content workflow management
- Quality oversight

## Best Use Cases

- Finding trending news topics
- Evaluating story potential
- Creating article briefs
- Coordinating multiple content pieces
- Managing editorial workflow
- Dispatching writing tasks to agents

## Example Usage

```
> @manager Find trending tech news and create article assignments
> @manager Evaluate these story ideas for newsworthiness
> @manager Create a content plan for this week's blog posts
```

## Coordination Strategy

The Managing Editor will:
1. Search for trending topics using web search and research tools
2. Evaluate stories for relevance and reader interest
3. Create detailed article briefs with target audience and key points
4. Dispatch writing tasks to appropriate agents
5. Coordinate content production workflow

## Technical Expertise

- **Research Tools:** Web search, news aggregation, Wikipedia
- **Agent Tools:** Task delegation, agent coordination
- **Editorial Skills:** Story evaluation, assignment creation, workflow management
- **Planning:** Content strategy, editorial calendar, topic curation

## When to Use

Use the Managing Editor when you need:
- News topic discovery
- Editorial coordination
- Content planning and strategy
- Multi-agent content workflows
- Story evaluation and assignment
- High-level content management
