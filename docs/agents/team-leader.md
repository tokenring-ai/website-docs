# Team Leader

**Agent ID:** `teamLeader`  
**Application:** TokenRing Coder  
**Category:** Planning & Management

## Overview

The Team Leader orchestrates full-stack projects by coordinating specialized agents, managing workflow, and ensuring cohesive implementation across all aspects of development.

## Capabilities

- Project planning and coordination
- Agent orchestration and delegation
- Full-stack feature implementation
- Workflow management
- Cross-functional integration
- Progress tracking and reporting

## Best Use Cases

- Large, multi-component features
- Full-stack application development
- Projects requiring multiple specialists
- Complex feature implementations
- End-to-end project delivery

## Example Usage

```
> @teamLeader Create a complete user authentication system with OAuth
> @teamLeader Build a dashboard with real-time analytics
> @teamLeader Implement a payment processing feature with Stripe integration
```

## Coordination Strategy

The Team Leader will:
1. Analyze requirements and break down tasks
2. Delegate to appropriate specialist agents
3. Coordinate between frontend, backend, and database work
4. Ensure consistent implementation across components
5. Manage testing and deployment considerations

## When to Use

Use the Team Leader when you need:
- Comprehensive feature development
- Multi-agent coordination
- Full-stack implementations
- Complex project management
- End-to-end delivery oversight
